## Milestone Summary

**Key Decisions**

- Keep the local Apple embedding path as the canonical gate and treat the remote GPU box as experiment-only.
- Treat `generated/` as runtime output and keep compact baselines/specs under tracked references.
- Preserve the stable script-wrapper interface while moving real implementation into `src/pdfmd`.
- Promote the challenge corpus as a hard non-regression gate without rebasing the canonical `why-ethics` holdout.
- Promote the repaired chapter-5 manual packet from report-only evidence into the canonical gate.

**Issues Resolved**

- The repo now has a real GSD planning surface and a canonical operator interface.
- The package split and wrapper parity are explicit and tested.
- The canonical `why-ethics` holdout is green and stable across repeated runs.
- The cross-book challenge corpus is enforced with `Specters of Marx` preserved as the clean negative control.
- The deferred `why-comment` `7c/7d` defect is fixed and enforced in the canonical gate.

**Issues Deferred**

- Real remote embedding comparisons on the SSH/GPU host are still experiment-only and were not promoted into the canonical path.
- `make map` can still be blocked by external Codex skill-loader YAML issues outside this repo.

**Technical Debt**
